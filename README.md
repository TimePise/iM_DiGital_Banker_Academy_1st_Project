# iM Digital Banker Academy 1st Project

## 📌 프로젝트 개요
**iM Digital Banker Academy 6기** 첫 번째 팀 프로젝트로, 기업 금융 데이터를 활용한 데이터 분석 프로젝트입니다.   
본 프로젝트는 금융 데이터의 전처리부터 탐색적 데이터 분석(EDA), 통계적 가설 검증, 고객 세분화까지 전 과정을 수행하여  
**데이터 기반 금융 의사결정 역량**을 강화하는 것을 목표로 합니다.

## 📊 주요 분석 내용

### 1. 데이터 전처리 (`전처리 코드.ipynb`)

#### 전처리 과정
```python
def pre_process(df:  pd.DataFrame) -> pd.DataFrame:
    # 1. 결측치 처리
    # 2. 날짜 데이터 변환 (YYYYMM → Period)

```

#### 주요 파생변수
- **총거래건수**: 모든 채널 거래건수 합계
- **총잔액**: 예금 잔액 합계
- **여신잔액**: 운전자금 + 시설자금 대출 합계
- **여신이용률**: 여신잔액 / 여신한도금액
- **디지털비중**: 디지털 거래건수 / 총거래건수
- **예대마진**: (여신잔액 × 5%) - (총잔액 × 2%)

### 2. 기초 EDA (`기초 EDA코드.ipynb`)

#### 업종별 분석
- **총잔액(AUM) 구성**:  예금 + 상품 잔액 분석
- **회전율**:  (입금+출금) / 예금잔액
  - 높은 회전율:  도매·소매, 음식·숙박, 운수·물류
  - 낮은 회전율: 부동산, 교육, 공공기관
- **상품 보유율**: 수익증권, 신탁, 퇴직연금 보유 비율

#### 고객등급별 분석
- 등급별 평균 잔액 및 거래 패턴
- 등급별 상품 보유 특성
- 등급별 채널 이용 특성

### 3. 가설 1:  업종별 예대마진 차이 (`가설1 코드.ipynb`)

#### 가설
> "업종별로 예대마진에 유의한 차이가 있는가?"

#### 분석 방법
1. **등분산성 검정**:  Levene Test (p < 0.05, 위배)
2. **Welch ANOVA**: F(17. 00, 19924.67) = 813.06, p ≈ 0
3. **결론**: 업종 간 예대마진 차이 유의함 (귀무가설 기각)

#### 주요 발견
| 구분 | 업종 | 평균 예대마진 |
|------|------|---------------|
| **손실 업종** | 금융 및 보험업 | -688.73 |
| | 보건업 및 사회복지 서비스업 | -12.64 |
| **우량 업종** | 예술·스포츠·여가관련 서비스업 | 257.55 |
| | 숙박 및 음식점업 | 187.33 |
| | 부동산업 | 169.05 |
| **제조업** | 제조업 | 91.17 |

#### 손실 업종 심층 분석
**부산광역시 금융업 특성**
- 기간별 예대마진 추이 분석
- 특정 고객의 대규모 예금 증가로 인한 마진 악화
- 여신 없이 예금만 보유한 "조달형 고객" 집중

### 4. 가설 2: 고객등급별 예대마진 차이 (`가설2 코드.ipynb`)

#### 가설
> "고객등급별로 예대마진에 유의한 차이가 있는가?"

#### 분석 방법
1. **등분산성 검정**:  Levene Test (p < 0.05, 위배)
2. **Welch ANOVA**: F(2.00, 206653.37) = 1095.18, p ≈ 0
3. **결론**: 등급 간 예대마진 차이 유의함

#### 주요 발견
| 등급 | 평균 | 중앙값 | 표준편차 | 표본수 |
|------|------|--------|----------|--------|
| **일반** | 16.83 | 2.69 | 76.66 | 95,059 |
| **우수** | 26.88 | 13.32 | 56.04 | 110,923 |
| **최우수** | 129.67 | 60.00 | 1,216.63 | 141,317 |

#### 핵심 인사이트
- 등급↑ → 중앙값↑ (일반 2. 66 → 우수 13.00 → 최우수 57.58)
- **최우수 등급의 이질성**: 표준편차 1,231.1로 매우 큼
  - 고마진·고여신 우량 고객 존재
  - 저마진·고예금 조달형 고객 혼재

| 지표 | 고마진·고여신 | 저마진·고예금 | 기타(최우수) |

1. **업종 분포** (χ² = 5,806.89, p ≈ 0)
   - **기타군**: 부동산·금융·숙박 비중↑
   - **저마진군**: 건설·도소매·전문기술 비중↑

2. **지역 분포** (χ² = 3,320.98, p ≈ 0)
   - **과다**:  서울·부산·경기·인천 (+)
   - **과소**: 대구·경북 (–)

3. **전담고객 여부** (χ² = 501.99, p ≈ 0)
   - **기타군**: 비전담 비율 7.9% (가장 높음)

## 💡 전략적 시사점

### 1. 업종별 전략

#### 손실 업종 관리
- **금융·보험업, 보건업**:  RM 배정 효율화, 조달 기여도 vs 마진 트레이드오프 분석
- **협회·단체**:  모니터링 강화, 여신 활용 유도

#### 우량 업종 집중
- **부동산·숙박·예술**:  교차판매 및 장기 거래 유도
- **제조업**: 안정적 중간 수익원, 관계 유지

### 2. 고객등급별 전략

#### 최우수 등급 세분화 관리

**고마진·고여신 그룹** (89,222건)
- 핵심 수익원 → **VIP 관리 강화**
- 높은 여신이용률 → 한도 관리 및 리스크 모니터링
- 디지털 채널 활성화 (50회↑ 42.2%)

**저마진·고예금 그룹** (35,763건)
- 조달 안정성 기여 → **예금 유지 전략**
- 여신 교차판매 기회 → 컨설팅 강화
- 창구 의존도 높음 → 디지털 전환 유도

**기타(최우수) 그룹** (16,296건)
- **가장 큰 개선 여지**
  - 비전담 비율 7.9% → RM 배정 우선순위
  - 디지털 0회 26.2% → 채널 활성화 캠페인
  - 수도권·부울경 집중 → 지역별 맞춤 전략
  - 부동산·금융·숙박 업종 많음 → 업종 특화 상품 제안

### 3. 채널 전략

#### 디지털 전환 우선순위
1. **기타(최우수)**: 디지털 0회 26.2% → 최우선 전환 대상
2. **저마진·고예금**: 창구 의존도↑ → 단계적 유도
3. **고마진·고여신**: 이미 디지털 활성화 → 고도화
